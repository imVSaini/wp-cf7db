LeadSync - Save Contact Form 7 submissions (including multi-step) into a custom database table with modern admin UI, column management, and CSV export.

== Description ==

LeadSync is a comprehensive Contact Form 7 database management solution that automatically saves all form submissions to a custom database table. Built with modern web technologies, it provides an intuitive admin interface for managing, filtering, and exporting form data.

= Key Features =

* **Automatic Form Storage**: Automatically saves all Contact Form 7 submissions to a custom database table
* **Modern Admin Interface**: Clean, responsive React-based admin interface for managing submissions
* **Column Management**: Show/hide and reorder columns to customize your data view
* **Advanced Filtering**: Search and filter submissions by form, date range, and field values
* **CSV Export**: Export submissions with custom column configuration and filtering
* **Multi-step Form Support**: Full support for Contact Form 7 multi-step forms
* **CFDB7 Migration**: Easy migration from Contact Form CFDB7 plugin
* **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
* **Security**: Built with WordPress security best practices

= Perfect For =

* Lead generation and management
* Contact form data collection
* Customer inquiry tracking
* Form submission analytics
* Data export and reporting
* Multi-step form handling

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/leadsync/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure Contact Form 7 is installed and active
4. Navigate to 'LeadSync' in your WordPress admin menu
5. Start managing your form submissions immediately

== Frequently Asked Questions ==

= Does this work with Contact Form 7 multi-step forms? =

Yes, LeadSync fully supports multi-step forms and captures all submission data.

= Can I migrate from Contact Form CFDB7? =

Yes, LeadSync includes built-in migration tools to transfer all your existing data.

= Is my data secure? =

Absolutely. All data is stored in your WordPress database with proper security measures.

= Can I customize the admin interface? =

Yes, you can show/hide columns, reorder them, and customize the display.

= Can I export my data? =

Yes, you can export submissions to CSV with custom column configuration and filtering.

= Is LeadSync mobile-friendly? =

Yes, the admin interface is fully responsive and works on all devices.

== Privacy Policy ==

LeadSync stores form submission data in your WordPress database. No data is sent to external servers. You maintain complete control over your data and are responsible for ensuring compliance with applicable privacy laws and regulations.

Data Stored:
* Form submission data (as configured)
* IP addresses (optional)
* User IDs (if logged in)
* Submission timestamps
* Form metadata
